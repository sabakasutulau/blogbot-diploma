# Выбор технологий разработки

## Python

Python выбран как основной язык разработки, потому что он позволяет быстро реализовать серверную логику, Telegram-бота, работу с базой данных и автоматизированные тесты.

## FastAPI

FastAPI используется для создания REST API и веб-приложения. Он автоматически формирует документацию API, что удобно для демонстрации проекта и тестирования запросов.

## Aiogram

Aiogram используется для реализации Telegram-бота. Библиотека поддерживает асинхронную обработку сообщений, состояния диалога и inline-кнопки.

## SQLAlchemy и PostgreSQL

SQLAlchemy обеспечивает работу с реляционной базой данных через ORM-модели. PostgreSQL выбран как основная база данных для итогового варианта проекта, так как это распространённая промышленная СУБД.

## HTML, CSS, JavaScript

Эти технологии используются для публичного сайта и панели администратора. JavaScript выполняет запросы к REST API без перезагрузки страницы.

## Pytest и Postman

Pytest используется для автоматизированной проверки API. Postman-коллекция нужна для ручной демонстрации и проверки запросов на защите.
