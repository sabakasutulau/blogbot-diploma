# Установка и внедрение программного продукта

## Минимальные требования

- Операционная система: Windows 10/11, Linux или macOS.
- Python 3.11 или выше.
- 2 ГБ оперативной памяти для локального запуска.
- 500 МБ свободного места на диске.
- Доступ к интернету для работы Telegram-бота.
- PostgreSQL 15+ для варианта с полноценной базой данных.

## Локальная установка

1. Распаковать архив проекта.
2. Открыть терминал в папке проекта.
3. Создать виртуальное окружение:

```bash
python -m venv .venv
```

4. Активировать окружение:

```bash
.venv\Scripts\activate
```

Для Linux/macOS:

```bash
source .venv/bin/activate
```

5. Установить зависимости:

```bash
pip install -r requirements.txt
```

6. Создать файл `.env` на основе `.env.example`.
7. Запустить веб-приложение:

```bash
uvicorn app.main:app --reload
```

## Запуск с PostgreSQL

1. Запустить базу данных:

```bash
docker compose up -d db
```

2. В `.env` указать строку подключения:

```env
DATABASE_URL=postgresql+psycopg://blogbot:blogbot@localhost:5432/blogbot
```

3. Запустить приложение:

```bash
uvicorn app.main:app --reload
```

## Запуск Telegram-бота

1. Создать бота через BotFather.
2. Указать токен в `.env`:

```env
BOT_TOKEN=<токен_из_BotFather>
```

3. Запустить бота:

```bash
python -m app.bot_runner
```

## Проверка после установки

1. Открыть `http://127.0.0.1:8000/api/health`.
2. Открыть `http://127.0.0.1:8000/docs`.
3. Создать публикацию через `/admin`.
4. Опубликовать публикацию.
5. Проверить отображение на главной странице.
6. Проверить команды Telegram-бота.
