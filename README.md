# BlogBot Diploma

Дипломный проект по теме: **«Разработка телеграм-бота для создания и ведения блогов»**.

Проект представляет собой систему для создания, хранения и публикации блоговых записей через Telegram-бота. Пользователь создает публикацию в Telegram, запись сохраняется в базе данных и после публикации отображается в веб-интерфейсе.

## Возможности

- создание публикаций через Telegram-бота;
- сохранение публикаций в базе данных;
- работа со статусами записей;
- публикация материалов в веб-интерфейсе;
- административная панель;
- REST API;
- Swagger-документация API;
- автоматизированные тесты.

## Стек технологий

- Python;
- FastAPI;
- Aiogram;
- SQLAlchemy;
- SQLite;
- Jinja2;
- HTML/CSS;
- Pytest;
- Uvicorn.

## Структура проекта

- `app/main.py` - backend-приложение FastAPI;
- `app/bot.py` - Telegram-бот;
- `app/bot_runner.py` - запуск Telegram-бота;
- `app/config.py` - настройки приложения;
- `app/db.py` - подключение к базе данных;
- `app/models.py` - модели базы данных;
- `app/schemas.py` - схемы данных;
- `app/crud.py` - операции с публикациями;
- `app/templates/` - HTML-шаблоны;
- `app/static/` - статические файлы;
- `tests/` - автоматизированные тесты.
